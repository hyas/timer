# Timer

A lightweight and customizable desktop timer built with Java.

## Features

- Countdown & Count-up Timer
- Transparent Overlay Mode
- Keyboard Shortcuts
- Dynamic Timer Events
- Customize Timer Colors
- Multiple Languages
- Light & Dark Themes
- Lightweight & Fast
- Cross-platform Support
